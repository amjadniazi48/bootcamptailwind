This layout uses only Tailwind's utility classes — no separate CSS file at all.

Resize your browser window: below Tailwind's md breakpoint (768px) the layout switches to flex-col and the sidebar stacks on top of the main content. At md and above, md:flex-row takes over and they sit side by side again — all without writing a single @media rule.
